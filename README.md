A lightweight Java Swing application for managing student records.
Features include adding, editing, deleting, and viewing students in a clean table interface.

Features

Add new students

Edit existing students

Remove students

Clear all records

Dynamic table view

How to Run (Eclipse)

Extract the ZIP file.

Open Eclipse → File → Import → Existing Projects into Workspace.

Select the extracted folder.

Open StudentManagementGUI.java → Run As → Java Application.

How to Run (Command Line)
javac -d bin src/StudentManagementGUI.java
java -cp bin StudentManagementGUI

Project Structure
src/
 └── StudentManagementGUI.java
